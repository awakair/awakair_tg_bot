# [@awakair_bot](https://t.me/awakair_bot)
Reimplementation of [@inline_predictions_bot](https://t.me/inline_predictions_bot)([github](https://github.com/awakair/inline_predictions_bot))

# How to run?
Firstly you need to install docker and docker compose and then do something like that
```code
export BOT_TOKEN=<your token>
docker compose build
docker compose up
```
